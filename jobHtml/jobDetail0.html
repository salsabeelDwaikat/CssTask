Front-end Developer

The Air Filter Company

1mo ago • Part Time • Worldwide

Job Description

The Air Filter Company is seeking a Front-end Developer to join their team. This part-time position offers the opportunity to work remotely on a variety of exciting projects. You will work closely with the design team to create and maintain interactive user interfaces using React and Sass.

Responsibilities

  • Develop and maintain front-end applications using JavaScript and React
  • Work closely with UX/UI designers to ensure high-quality user experiences
  • Implement responsive designs and optimize performance for mobile and desktop
  • Collaborate with back-end developers to integrate front-end with back-end systems
  • Write clean, reusable, and maintainable code

Requirements

  • Experience with JavaScript and React
  • Strong knowledge of HTML, CSS, and Sass
  • Experience with version control systems such as Git
  • Ability to write clean, efficient, and reusable code
  • Strong problem-solving skills and attention to detail
